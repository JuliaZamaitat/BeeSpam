# BeeSpam


How to use:

1. Clone the project.
2. Change the path in sendMessage.txt according to your path
3. In your command line tool, run:
  ```
  /usr/bin/osascript /Users/<YourUser>/<YourPath>/BeeSpam/sendMessage.txt <phone-number>
  ```
